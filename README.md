# Setup:
1) **Open Visual Studio Code and -file-open folder- select folder**
2) **Open the terminal and enter the following codes in order:**
  **pip install flask**
   **pip install bcrypt**
! *If pip doesn't work:  python -m pip install flask bcrypt*
4) **Restart Visual Studio Code, then run the app.py file while pressing F5.**
